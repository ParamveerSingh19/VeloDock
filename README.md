# VeloDock 📂

A simple, lightweight, and **efficient** Command-Line File Manager built using **Python**. This project allows users to efficiently create, view, read, update, and delete files through an interactive command-line interface while demonstrating the fundamentals of Python file handling and exception handling.

---

## 🚀 Features

- 📄 Create new files

- 📂 View all files in the current directory

- 🗑️ Remove existing files

- 📖 Read the contents of a file

- ✏️ Update files by appending new content

- ⚠️ Handles file-related exceptions gracefully

- 💻 Interactive menu-driven command-line interface

- ⚡ Lightweight application using only Python's built-in libraries

---

## 🛠️ Tech Stack

- **Python 3**

- **OS Module**

- **File Handling**

- **Exception Handling**

---

## 📂 How to Use

1. **Clone the repository**:

   ```bash
   git clone https://github.com/ParamveerSingh19/VeloDock.git
   ```

2. **Navigate into the project folder**:

   ```bash
   cd VeloDock
   ```

3. **Run the application**:

   ```bash
   python app.py
   ```

4. **Select an option** from the menu to perform file operations.

No additional setup or external libraries are required — it works directly with Python 3.

---

## 📋 Available Operations

- **Create File:** Generate a new file in the current directory.

- **View Files:** Display all files available in the current directory.

- **Remove File:** Delete an existing file.

- **Read File:** View the contents of a selected file.

- **Update File:** Append new content to an existing file.

- **Exit:** Close the application.
